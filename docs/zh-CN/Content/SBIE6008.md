# SBIE6008

**信息** SBIE6008 沙箱 %2 的配置 %3 需要支持者证书，否则无法使用

**严重性**: 错误（弹窗）

## 说明

当沙箱配置使用了需要支持者证书的功能，而系统中没有有效证书允许该配置使用时，会显示此错误。与 SBIE6004 不同，SBIE6004 只是提示证书需求并给予宽限期，而 SBIE6008 表示没有证书完全无法使用该配置

## 常见原因

- 沙箱配置启用了一个或多个受支持者证书控制的功能（如特定加密或安全模式设置）
- 未安装支持者证书，或者已安装的证书无效或未激活
- 证书缺少该配置所需的选项标志

## 建议操作

1. 检查消息中引用的配置（`%3` 为配置标识符），如果不打算提供支持者证书，请禁用需要证书的功能
2. 安装并激活包含该配置所需选项标志的支持者证书
3. 安装证书后，请重启沙盘服务和驱动，以便检测到新证书
4. 如果该配置应当无需证书即可使用，请检查功能标志，并查看驱动/服务日志，查找证书检查失败的原因
