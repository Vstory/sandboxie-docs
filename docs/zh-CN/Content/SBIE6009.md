# SBIE6009

**消息**

SBIE6009 沙箱 %2 的配置 %3 需要高级支持者证书

**严重性**：错误（弹出窗口）

## 描述

当沙箱配置需要高级支持者证书（比普通支持者证书具有更高级的权限集）时，会显示此弹出错误。如果没有有效且激活的高级证书，则该配置无法使用。

高级证书用于启用进一步放宽沙箱隔离或允许特权代理操作的功能。例如高级加密处理、网络代理或项目为支持者保留的其他敏感功能。

## 常见原因

- 沙箱配置启用了需要高级支持者证书的功能
- 未安装高级证书，或已安装的证书未激活或缺少所需的高级选项标记
- 驱动程序/服务未能检测或验证已安装的证书

## 建议操作

1. 检查消息中提到的配置（`%3` 为配置标识符），如果不打算提供高级证书，则禁用需要高级证书的功能
2. 如果需要高级功能，请获取并安装高级支持者证书，并确保其已激活且包含所需的选项标记
3. 在安装证书后，重启沙盘组件（服务和驱动），以便驱动能检测到更改
4. 如果您确信证书已正确安装，但消息仍然出现，请启用驱动/服务追踪，并通过日志检查证书查询结果以进行故障排查
