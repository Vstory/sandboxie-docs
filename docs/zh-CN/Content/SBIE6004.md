# SBIE6004

**信息** SBIE6004 沙箱 %2 的配置 %3 需要支持者证书。该沙箱中的进程将在 5 分钟后终止。如需长时间使用，请考虑支持本项目。

**严重性**: 错误（弹窗）

## 描述

当 沙盘 检测到某个沙箱配置需要支持者证书（或高级支持者证书）但当前未安装或未激活此证书时，会显示此错误。该信息会以弹窗形式展示，并可能导致受影响沙箱中的进程在短暂宽限期后被终止。

典型场景包括降低隔离强度的功能或启用特权代理操作（如 EFS 代理），这些功能通常依赖支持者证书选项。

## 常见原因

- 沙箱配置中启用了需要支持者证书的功能（例如 `EnableEFS`），但系统中没有有效且已激活的支持者证书
- 已安装证书存在，但未被标记为激活状态，或者缺少所需的选项标志（例如用于 EFS 代理的加密标志）
- 从驱动或服务查询证书时，证书验证失败

## 建议操作

1. 检查消息中提到的沙箱配置（`%3` 是配置名称，`%2` 是沙箱名称）。如果不打算提供证书，请禁用需要证书的相关功能
2. 如果确实需要使用该功能，请安装并激活包含所需选项标志的支持者证书
3. 安装证书后，重启沙盘组件（服务/驱动），以便驱动和服务能够检测到新证书
4. 若在已有有效证书的情况下仍然看到该消息，请收集日志（启用跟踪），并检查服务/驱动证书查询路径是否有错误
